A distortion shader material.

Shader code taken from Martins Upitis' water/underwater blend: http://devlog-martinsh.blogspot.de/2013/09/waterunderwater-sky-shader-update-02.html

Parameters:

  • [options] Object optional
    • The options.
    • [perturbMap] Texture optional
      • If none is provided, the shader will generate perlin noise on the fly.
    • [rollOffSpeed] Vector2 optional
      • The water roll off speed. X affects the overall roll off, while Y controls the droplets.
    • [waveStrength] Vector2 optional
      • The distortion wave strength. X = sine, Y = cosine.
    • [color] Color optional
      • The droplet tint.
